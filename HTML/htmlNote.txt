HTML
<html>标签成对出现，开始标签称为起始标签，结束标签称为闭合标签。

每种标签有自己的意义，标签不区分大小写，一般默认全小写。

html文件结构
	<html>称为根标签，所有网页标签都在其内

html语义化：
	在什么情况下我可以使用什么标签才合理。
	a. 更容易被搜索引擎收录
	b. 更容易读出网页内容(例如打开网页速度)

<!DOCTYPE html> 
	声明文档类型，不是html标签
	其必须放在HTML文档的第一行
	这是HTML5的文档说明，不同版本HTML文档说明格式不同

<meta>标签可以决定文档编码	
	<meta charset="utf-8">
	在<head>标签内
	
<p>段落标签
	在<body>标签内定义
	<p>是块级元素，浏览器会自动加换行，添加空行。不加闭合标签大多数浏览器会默认加上，但是禁止这样做
	所有标签内连续的空格或者换行浏览器会自动省略
	
<hx>标题标签
	在<body>标签内定义
	<h1>~<h6>来定义，<h1>定义最大标签，<h6>定义最小标签
	不要仅仅为了文字加粗和变大就把段落的标签改为标题标签，这样不符合html语义化
	<hx>块级元素，自动换行，添加空行
	
<em>，<strong>强调标签
	内敛标签，不自动换行加空行
	<em>斜体显示
	<strong>加粗显示
	
<q>短引用标签
	浏览器将标签内容自动加上引号，此处关键点是在于<q>标签的语义化
	
<blockquote>长引用标签
	浏览器会自动缩进标签内内容，并且开始结束行自动换行

<br/>换行标签
	标签内无内容，直接使用
	<br>与<br/>没有区别，但是为了语义，建议使用<br/>
	标签内无内容，称为空标签

</hr>分隔线标签
	空标签
	自动换行并加上分隔线，CSS可以修改分割线的样式
	<hr>与<hr/>的关系参考<br>与<br/>一样

	
特殊字符
	&nbsp; 空格
	&reg; 已注册
	&copy; 版权
	&trade; 商标

<address>地址标签
	标签内内容以斜体展示
	
<code>代码标签
	单行代码使用<code>标签，字体改变，换行

<pre>标签
	多行代码，字体改变
	也叫预显示标签，不会省略文档中的空格换行等符号。
	不只是为了显示多行代码
	
列表标签
	无序列表：以黑圈圈表示
	<ul>
		<li></li>
		<li></li>
	</ul>
	每个<li>标签内部可以使用段落<p>，换行符<br/>，图片，链接以及其他列表等

	有序列表：以数字表示
	<ol>
		<li></li>
		<li></li>
	<ol>
	
	
	自定义列表
	<dl>自定义列表的开始
		<dt>每一项的定义
			<dd>每一行的定义</dd>
		</dt>
	</dl>
	
HTML块
	HTML块元素
		块级元素，以新行开始，新行结束：<hx> <p> <ul> <table> <div>等
		内联元素，不会以新行开始和结束：<em> <strong> <a>等
	
	HTML <div class/id>：可以定义文档中的分区或节，使用class或id来标记div可以使div作用更有效
		class属性可以有多个，而id属性只有一个

	HTML <span>：
		<span>被用来组合HTML中行内元素
		<span>标签没有固定格式表现，当采用样式(css)，它才会有视觉上的改变
		也可以使用class，id。class，id只要与css搭配使用
